## 安装docker
在宝塔软件商店中搜索 docker 点击安装。
![](https://zt0729-picture-bed.oss-cn-beijing.aliyuncs.com/uTools_1643184300787.png)

## 拉取青龙镜像
在终端上输入
```
docker pull whyour/qinglong
```

## 启动青龙
启动代码，直接复制粘贴到终端，回车
```
docker run -dit \
   -v $PWD/ql/config:/ql/config \
   -v $PWD/ql/log:/ql/log \
   -v $PWD/ql/db:/ql/db \
   -v $PWD/ql/scripts:/ql/scripts \
   -p 5700:5700 \
   --name qinglong \
   --hostname qinglong \
   --restart always \
   whyour/qinglong:latest
```
用代码可以查看运行状态
```
docker ps
```
![](https://zt0729-picture-bed.oss-cn-beijing.aliyuncs.com/uTools_1643184856023.png)

## 登陆青龙面板
先去服务器控制面板开放 5700 端口，在浏览器输入 ip:5700，这里的 ip 是你服务器的公网 ip。
