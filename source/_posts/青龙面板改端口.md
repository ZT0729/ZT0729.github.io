最近青龙面板，一直被别人撞库，导致密码错误次数过多，要等个几十万秒之后登录，烦死了，所以更改下青龙面板的端口。

1. 先连接终端，获得所有容器的基本信息，并记住容器id
```
sudo docker ps -a
```
![](https://zt0729-picture-bed.oss-cn-beijing.aliyuncs.com/uTools_1642441782225.png)
2. 执行停止容器命令
``` 
docker stop 上面获取到的容器id 
```
3. 停止docker
```
 sudo systemctl stop docker 
```
4. 进入容器配置文件夹，注意容器id后补上*号自动补全
```
cd /var/lib/docker/containers/容器id*
```
5. 用vim修改配置文件，修改端口
```
vi hostconfig.json
```
6. 进入编辑页面后，按i进入修改状态，将"HostPort":"5700"改成"HostPort":"xxxx"，前面的5700/tcp别动。云服务安全组开放你设置的端口
7. 改完按esc退出编辑状态，输入 :wq 保存退出
8. 启动docker
```
sudo systemctl start docker
```
9. 重启容器即可
```
docker start 上面获得的容器id
```
